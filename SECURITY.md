# Security Policy

## Supported Versions

Only the latest version of the FOC Companion is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of FOC Companion seriously. If you believe you have found a security vulnerability, please report it privately to us.

**Please do not report security vulnerabilities through public GitHub issues.**

To report a vulnerability, please send an email to goran.obradovic@gmail.com with:
- A description of the vulnerability
- Steps to reproduce it
- Potential impact

We will attempt to acknowledge your report within 48 hours and provide a fix as soon as possible.

Thank you for helping keep FOC Companion secure!
